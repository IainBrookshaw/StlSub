\begin{DoxyCompactList}\small\item\em The \char`\"{}\-Mesh\char`\"{} class is the master class for the S\-T\-L meshes. It includes the original user input mesh, methods to load this and subdivide it into a finer stl mesh. The subdivision does not alter the overall geometry, this is not a smoothing utility. It is also important to note that the exact size of the final triangles is undefined, only the maximum triangle size is specified. \end{DoxyCompactList}\end{DoxyCompactItemize}
